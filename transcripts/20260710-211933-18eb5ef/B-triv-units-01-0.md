**1 gibibyte (GiB) = 1,024 mebibytes (MiB).**

This is because both units use binary prefixes (powers of 1024):
- 1 GiB = 2^30 bytes
- 1 MiB = 2^20 bytes
- 2^30 ÷ 2^20 = 2^10 = 1,024
