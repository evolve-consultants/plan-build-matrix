HTTP status code **418** means **"I'm a teapot"**.

This is a genuine (if whimsical) status code defined in [RFC 2324](https://tools.ietf.org/html/rfc2324), an April Fools' RFC from 1998. It's intended as a response when a server receives a request to brew coffee using the Hyper Text Coffee Pot Control Protocol (HTCPCP) — a satirical protocol for controlling networked coffee makers.

In practice, some servers return 418 as an easter egg or to indicate that a request is nonsensical or outside the server's capabilities. It's rarely used seriously, but it's a valid HTTP status code and some web frameworks (like Node.js's Express) support it.
