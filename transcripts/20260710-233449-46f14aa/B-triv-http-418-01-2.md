HTTP status code **418 I'm a teapot** is a joke/easter egg status code defined in RFC 2324 (April 1, 1998). It was intended as an April Fools' RFC and indicates that the server refuses to brew coffee because it is, permanently, a teapot.

In practice, you'll almost never encounter it in the wild. Some servers (like Google's) return it as a humorous easter egg, but it has no real-world use case.
