# HTTP Status Code 418

**418 I'm a Teapot**

This is a humorous status code that indicates the server refuses to brew coffee because it is a teapot.

## Details

- **RFC**: Defined in [RFC 2324](https://tools.ietf.org/html/rfc2324) (April 1, 1998)
- **Type**: Client error
- **Purpose**: A joke/Easter egg status code

## Context

RFC 2324 was an April Fools' RFC about the "Hyper Text Coffee Pot Control Protocol" (HTCPCP). The 418 status code was meant to be used when a coffee pot receives a request to brew coffee but cannot because it's actually a teapot.

## Real-World Usage

Despite being a joke, some developers have implemented it as an Easter egg in their applications. A few notable uses:
- **Google's teapot**: google.com/teapot returns 418
- Some APIs use it humorously in their error responses

In practice, you'll almost never encounter this status code in legitimate web applications—it's purely for fun and novelty purposes.
