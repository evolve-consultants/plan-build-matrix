# HTTP Status Code 418

**418 I'm a teapot** is a humorous status code that indicates the server refuses to brew coffee because it is a teapot.

## Details

- **Type:** Client error (4xx)
- **Official source:** RFC 2324 (April 1, 1998)
- **Actual use:** This is an April Fools' joke RFC and isn't meant for real-world use

## Context

The code was defined in a playful RFC about the Hyper Text Coffee Pot Control Protocol (HTCPCP). It's never actually used in production—it's just a lighthearted Easter egg that some servers or applications occasionally return as a joke.

If you encounter 418 in the wild, it's almost certainly intentional humor rather than a genuine server error.
