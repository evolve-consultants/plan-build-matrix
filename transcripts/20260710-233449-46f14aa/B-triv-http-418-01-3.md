HTTP status code **418** means **I'm a teapot**.

This is an April Fools' joke status code defined in [RFC 2324](https://tools.ietf.org/html/rfc2324) (published April 1, 1998). It's returned when a coffee pot receives a request to brew coffee. The accompanying response body typically contains a short poem.

In practice, it's almost never used in real applications — it exists purely as a joke and Easter egg. Some services have implemented it humorously (notably Google's old cloud services and some indie projects), but you won't encounter it in standard web traffic.
