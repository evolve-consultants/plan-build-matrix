HTTP status code **418** means **I'm a Teapot**.

This is an April Fools' joke status code defined in [RFC 2324](https://tools.ietf.org/html/rfc2324) (April 1, 1998). It's returned when a server refuses a request to brew coffee because it is, in fact, a teapot—not a coffee machine.

In practice, it's almost never used in real applications. Some servers or APIs occasionally return it as a humorous easter egg, but it has no legitimate use case. It's not a standard HTTP status code you'll encounter in normal web development.
