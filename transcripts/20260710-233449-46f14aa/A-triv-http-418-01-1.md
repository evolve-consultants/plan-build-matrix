# HTTP Status Code 418

**418 I'm a teapot**

This is a humorous status code that indicates the server refuses to brew coffee because it is a teapot.

## Background

It's defined in [RFC 2324](https://tools.ietf.org/html/rfc2324), an April Fools' RFC from 1998 about the Hyper Text Coffee Pot Control Protocol (HTCPCP). The joke was that if a client tried to brew coffee using HTTP on a teapot device, the teapot would respond with 418.

## Real-World Usage

While created as a joke, some developers have actually used it in practice:
- As a playful Easter egg in web applications
- To indicate that a request doesn't make sense for that particular server
- In protest against surveillance or as a symbolic response (famously used in some cases)

**In practice**, you'll almost never encounter this status code. It's not part of standard HTTP usage and has no real functional purpose.
