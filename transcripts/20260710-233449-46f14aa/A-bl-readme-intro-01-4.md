## Tips for filling this in:

- **The hook:** Finish this sentence: "This project helps you..." or "This project is for people who..."
- **Benefits over features:** Replace technical specs with "what does it let me do?"
- **The example:** One copy-paste block that actually works, not a 10-step setup
- **Tone:** Imagine explaining it to a colleague over coffee, not a board meeting
