HTTP status code **418** means **"I'm a teapot."**

It's a joke status code defined in RFC 2324 (April 1, 1998) as part of an April Fools' RFC. The spec humorously states that any attempt to brew coffee with a teapot should result in this response. 

Some real-world implementations (notably Google's servers and some JavaScript frameworks) have actually used it as an Easter egg. It's not intended for production use in serious applications, but it shows up occasionally in developer culture.
