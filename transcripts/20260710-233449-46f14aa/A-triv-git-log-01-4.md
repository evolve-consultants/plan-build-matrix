# Show last 5 commits, one per line

```bash
git log -5 --oneline
```

This will display the 5 most recent commits with their hash and commit message on a single line each.

**Example output:**
```
a1b2c3d Fix bug in authentication
e4f5g6h Add user profile page
i7j8k9l Update dependencies
m10n11o Refactor database queries
p12q13r Initial commit
```

You can adjust the number (`5`) to show more or fewer commits as needed.
