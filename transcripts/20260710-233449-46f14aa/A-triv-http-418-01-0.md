# HTTP Status Code 418

**418 I'm a teapot** is a humorous HTTP status code that means the server refuses to brew coffee because it is, permanently, a teapot.

## Details

- **Category**: Client error (4xx)
- **Origin**: Defined in [RFC 2324](https://tools.ietf.org/html/rfc2324) (April 1, 1998)
- **Nature**: **April Fools' joke** – this is not a standard status code for real-world use

## Context

The RFC was published on April 1st and describes the "Hyper Text Coffee Pot Control Protocol" (HTCPCP) as a playful extension of HTTP. It's meant to be humorous and isn't used in actual web applications.

That said, some websites have implemented it as an Easter egg or joke response.

**In practice**: You'll almost never encounter this in real web development. It's just a fun piece of internet history! 🫖
