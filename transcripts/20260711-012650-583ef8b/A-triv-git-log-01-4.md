# Show last 5 commits, one line each

```bash
git log -5 --oneline
```

This will display the 5 most recent commits with their hash and commit message on a single line each.

**Example output:**
```
a1b2c3d Fix bug in login
d4e5f6g Add user profile page
h7i8j9k Update dependencies
l0m1n2o Refactor database queries
p3q4r5s Initial commit
```
