# HTTP Status Code 418

**418 I'm a teapot** is a humorous status code that indicates the server refuses to brew coffee because it is a teapot.

## Context

This is an **April Fools' joke** status code defined in [RFC 2324](https://tools.ietf.org/html/rfc2324), published on April 1, 1998. The RFC humorously describes a protocol for controlling coffee pots via HTTP.

## Real-World Usage

- **Rarely used in production** — It's meant as a joke and has no practical application
- **Some websites use it playfully** — A few sites return 418 as an Easter egg or for humorous purposes
- **Not a standard error code** — You won't encounter it in normal web browsing

## Similar Joke Status Codes

Other humorous HTTP codes include:
- **451 Unavailable For Legal Reasons** — Though this one has actual legal purpose
- **999 Request Denied** — Used by some services

In practice, you'll almost never see 418 in real applications. If you do, the server is having fun!
