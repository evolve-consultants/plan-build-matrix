HTTP status code **418** means **I'm a teapot**.

It's a joke status code from [RFC 2324](https://tools.ietf.org/html/rfc2324), an April Fools' RFC published in 1998. It's returned when a server receives a request to brew coffee using the Hyper Text Coffee Pot Control Protocol (HTCPCP) — which doesn't actually exist.

In practice, you'll almost never see it in the wild. Some developers use it playfully in their APIs as an easter egg, but it has no real standardized meaning. If you encounter it, treat it as either a joke or a misconfiguration.
