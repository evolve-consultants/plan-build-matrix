HTTP status code **418 I'm a teapot** is a humorous response code defined in RFC 2324 (an April Fools' RFC from 1998). It's returned when a server receives a request to brew coffee using the Hyper Text Coffee Pot Control Protocol (HTCPCP) — which doesn't actually exist.

In practice, it's rarely used. Some servers or APIs return it as a joke or Easter egg, but it has no real-world application. If you encounter it in the wild, it's almost certainly intentional humor rather than a genuine error.
