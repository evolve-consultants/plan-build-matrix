HTTP status code **418** means **I'm a teapot**.

It's a humorous status code defined in RFC 2324 (an April Fools' RFC from 1998). It's returned when a server receives a request to brew coffee using the Hyper Text Coffee Pot Control Protocol (HTCPCP) — which was itself a joke specification.

In practice, it's almost never used in real APIs. Some sites return it as an Easter egg or as a way to humorously reject requests, but it has no standard semantic meaning for actual web applications.
