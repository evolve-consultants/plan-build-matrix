# Show last 5 commits, one line each

```bash
git log -5 --oneline
```

This will display the 5 most recent commits with their hash and commit message in a compact format.

**Related options:**
- `git log -10 --oneline` — show last 10 commits
- `git log --oneline -n 5` — alternative syntax (same result)
- `git log --oneline --graph -5` — add a branch visualization
