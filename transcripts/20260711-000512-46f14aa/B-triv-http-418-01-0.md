HTTP status code 418 is "I'm a teapot." It originated as an April Fools' joke in RFC 2324 (1998), a parody spec called the "Hyper Text Coffee Pot Control Protocol." The code indicates that a server refuses to brew coffee because it is, in fact, a teapot. It's not part of any official web standard for practical use, but some servers and frameworks implement it playfully, and it occasionally gets used as an easter egg or for humorous error handling.
